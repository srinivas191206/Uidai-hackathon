# UIDAI Analytics Command Center

An enterprise-grade operational intelligence dashboard for UIDAI.

### Key Features:
- **Demand Velocity Engine**: Real-time rate of change monitoring.
- **Closed-Loop Incident Response**: Automated anomaly detection and action tracking.
- **PSACI Index**: Pincode Service Access Concentration Index for equity analysis.
- **Enterprise Security Branding**: Hardened UI for government-grade presentations.

### Deployment Info:
- SDK: Docker
- Base Image: Python 3.9-slim
- Requirements: Streamlit, Pandas, Plotly, PySpark (for backend logic references)
